### POST /users/register

> Register

**Header**

```
not needed
```

**Body**

```
name          = <your name>
email         = <your email>
password      = <your password>
profile-image = <image file>
newsPrefId    = <news preference id>
```

_Response 201 Created_

```JSON
{
  "id": 1,
  "name": "<your name>"
}
```

_Response 400 Bad Request_

```JSON
[
  "<field> cannot be empty",
  "Invalid email format",
  "Email already exists"
]
```
